# Luxestay Hotel

Welcome to the Luxestay Hotel application! This is a React-based web application designed to provide users with a seamless experience for booking hotel rooms and exploring hotel services.

## Table of Contents

- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)

## Features

- Browse available hotel rooms
- Detailed room information
- User profile management
- Contact form for inquiries
- Responsive design

## Installation

To get started with the Luxestay Hotel application, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/luxestay-hotel.git
   ```
2. Navigate to the project directory:
   ```
   cd luxestay-hotel
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To run the application in development mode, use the following command:
```
npm run dev
```
Open your browser and navigate to `http://localhost:3000` to view the application.
